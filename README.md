# 🎯  Driver Drowsiness Detection System
# Project Overview

This project detects driver drowsiness using computer vision and deep learning techniques. It analyzes facial features such as eye closure and yawning behavior to determine whether a driver is alert or fatigued.

The system is designed to improve road safety by identifying early signs of drowsiness and triggering alerts before accidents occur. It uses image-based classification models trained on real-world facial datasets.


# Objectives
  * Detect driver drowsiness using facial indicators
  * Classify eye state (Open / Closed)
  * Classify mouth state (Yawn / No Yawn)
  * Combine predictions to determine fatigue level
  * Build deep learning models using CNN & transfer learning
  * Perform model evaluation using accuracy and confusion matrix
  * Improve road safety through early detection
